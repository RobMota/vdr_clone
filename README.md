#### Bibliotecas Usada:

- React
- Tailwindcss
- Node 20

#### Para usar o :

1. Abra o terminal ( terminal / command prompt(cmd) )
2. Clone o repositorio ( git clone https://github.com/RobMota/vdr_clone )
3. Acesse a pasta ( cd vdr_clone )
4. e escreva os comandos abaixo:

- `npm install`
- `npm run dev`
